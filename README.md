## ABOUT

This project showcases a scalable website built with Docker and Kubernetes. The application is containerized using Docker, ensuring consistency across different environments. Kubernetes is employed for orchestrating the deployment, scaling, and management of the application, allowing for robust performance and high availability. This setup demonstrates modern best practices in cloud-native application development and deployment.
